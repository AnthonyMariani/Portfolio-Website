NFL Analytics Dashboard

The NFL Analytics Dashboard is an innovative tool designed to distill comprehensive NFL player statistics from a CSV-sourced SQL database into user-friendly visual data, facilitating easy access to player performance metrics, fantasy football management, and in-depth statistical analysis through a sleek JavaFX interface.

Skills & Tools:

  • SQL Expertise: Demonstrated ability to structure and query MySQL databases for responsive data retrieval.
  • Database Design: Engineered a normalized relational database schema to efficiently manage and query NFL player statistics.
  • Java Proficiency: Utilized JDBC to integrate Java application logic with SQL database operations seamlessly.
  • Data Importing and Parsing: Automated the extraction and transformation of extensive NFL data from CSV files into the database.
  • User Interface Design: Developed with JavaFX to ensure a responsive and intuitive user interaction, emphasizing a clean and navigable design.
  • Analytics and Reporting: Implemented aggregation queries to present statistical data and performance metrics in a user-friendly format.

Project Insights:

  • Deployed dynamic SQL statements within Java methods to fetch real-time data, providing users with up-to-date insights.
  • Enabled complex data analysis features, like filtering NFL players by stats, through meticulous SQL query development.
  • Facilitated a fantasy football feature allowing users to simulate team management scenarios based on live data feeds.

As for the functionality, we have the Player Search tab that allows users to search stats by player, which will show them individual game statistics and season statistics. The All Players tab allows users to filter the database by position in order to find players. Third, we have the Leaderboards tab that allows users to filter the database by nearly any metric, showing the leaders in each respective statistic. Finally, we have the Fantasy tab that allows users to track player performance and PPR fantasy points across the season. This dashboard is a testament to my dedication to creating applications that not only process data efficiently but also present it in a manner that enriches user experience and decision-making. It's a clear indicator of my capability to handle and visualize data, underscoring my passion for technology, data analysis, and sports.
